Mobile Apps Spain
Aplicaciones Android & IOs  --> Web Services --> MongoDB

Mobile Apps Spain planea distribuir aplicaciones para móviles para que sus clientes realicen pedidos.

Tendrá que desarrollar un web service que escuche los pedidos gestionando la seguridad y los guarde en una base de datos MongoDB. Tendrá que contratar los servidores tanto para la los web services como para la BBDD.

Si la plicación tienen éxito tendrán que ser capaces de crecer de forma casi instantánea de un único servidor a múltiples servidores balanceados.

Escamarla Integration Platform
------------------------------

Con Escamarla sólo tendrá que configurar la seguridad de quién se puede conectar.

Automáticamente se generarán los servidores de web services y de la bbdd MongoDB a la que tendrán una cadena de conexión para acceder a ellos.

La carga de trabajo producirá el aumento automático de los servidores para que sean capaces de atender todas las peticiones recibidas, sin retrasos.

Cuando la carga de trabajo desciende la dimensión de los servidores bajará automáticamente.

Mobile Apps Spain no gastará dinero en el desarrollo de esos web services y pagrá sólo por el uso del servicio de Escamarla, pagando muy poco si hay poco uso y ese pago sólo crecerá si aumentan sus clientes.
